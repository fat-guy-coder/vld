指标说明

包体积 (KB)
框架核心包的 Gzip 压缩后大小。
速度评分
基于 Keyed 算法 benchmark 的综合速度评分 (1-5分，5分最快)。
内存 (MB)
创建1000行数据后，浏览器占用的堆内存大小。
FCP (ms)
First Contentful Paint (首次内容绘制): 浏览器渲染出第一个 DOM 元素所需的时间 (越低越好)。
LCP (ms)
Largest Contentful Paint (最大内容绘制): 视口中最大可见内容元素渲染完成所需的时间 (越低越好)。
INP (ms)
Interaction to Next Paint (下次绘制交互):衡量用户交互（如点击）到屏幕上出现视觉反馈的总时长，是核心的响应性指标 (越低越好)。
Signal Create (ops/s)
每秒可创建的响应式信号数量 (越高越好)。
Signal Update (ops/s)
每秒可更新的响应式信号数量 (越高越好)。
TBT (ms)
Total Blocking Time (总阻塞时间): 主线程被长任务阻塞的总时间，影响页面响应性 (越低越好)。
Hydration (ms)
将服务器端渲染的静态HTML转化为可交互应用所需的时间 (越低越好)。
